Requirements #

FriskyMouse is a WinForms application that requires NET 7 Desktop runtime to be installed.

  • Git — latest source release
  • Node.js — latest LTS version or newer
Why portable version? Doks uses npm (included with Node.js) to centralize dependency management, making it easy to update resources, build tooling, plugins, and build scripts.
